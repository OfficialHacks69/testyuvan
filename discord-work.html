Music Bot

Working on a Discord Music Bot to use on small servers with easy to use commands and also supports looking up lyrics for the song you're listening to!

Also supports a queuing set-up with votes skipping procedure to ensure all users have a pleasant time and also has looping system if you would like to listen to a specific song without any disturbance.

Uses discord.js and the YouTube APIto request the song from YouTube and finally uses a crucial NPM package named node-opus allowing you to hear the sound of the bot.
